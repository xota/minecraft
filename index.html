Commands en command blocks gebruiken in minecraft

Kaart voor Experts (meer dan 10 uur ervaring en zin om dingen zelf uit te zoeken), gemaakt door Enzo Raggani over Programmeren met tekst

In deze lesbrief leer je hoe je een command block kan krijgen, hoe je een command block gebruikt, en de basis commands

"Voordat we beginnen, moet je eerst controleren bij instellingen van je wereld of 'cheats activeren' en 'opdrachtblokken ingeschakeld' aanstaan. Als ze niet aanstaan, dan kan je geen command blokken krijgen.

Wat zijn commands?
Commands zijn woordjes die je kan invoeren in de chat, waarmee je een bepaalde functie (de)activeert. Je kan bijvoorbeeld met een command het weer veranderen.

Wat is een command block?
Command blocks zijn blokken waarin je commands in kan voeren, en kan activeren met verschillende materialen zoals een hendel, een knop of je kan het aan een redstone constructie koppelen. De commands worden pas uitgevoerd als de command block een signaal krijgt

Hoe gebruik je een command?
Klik op 't' om de chat te openen, vervolgens klik je op '/' en als het goed is zie je een lijst met allemaal woordjes. Je kan op 'TAB' klikken om elke command af te gaan, je ziet naast elk woordje een korte uitleg wat de command doet

Hier zijn de meest gebruikte handige commands:
/tp = hiermee kan je iemand laten teleporteren/
/gamemode = hiermee verander je je speltype, je hebt 4 opties. Survival, creative, default en adventure.
/gamerule = hiermee kan je bepaalde regels in je minecraft wereld wijzigen, die je niet kan wijzigen in de standaard instellingen.
/weather = hiermee kan je het weer bepalen.
/time set = hiermee bepaal je de tijd, dus of het overdag of nacht is.
/give = hiermee kan je blokken in een inventory spawnen
/summon = hiermee kan je een mob spawnen

Hoe krijg je een command block?
Typ in de chat '/give @s command_block'

Hoe gebruik je een command block?

Plaats de command block, en klik op je rechtermuisknop om je command block te 'openen'. Vervolgens krijg je de 'instellingen' van je command block. Je kan bij 'command invoer' elke command invoeren die je ook kan invoeren in de chat invullen, maar deze keer hoef je geen '/' ervoor te zetten.

Bij type blok kan je kiezen tussen 3 opties: Impulse, chain of repeat.
Bij impulse voert de command block de command 1 keer uit wanneer er een signaal is gegeven
Bij chain voert de command block de command continu uit zolang er maar 'stroom' is van redstone
Bij repeat voert de command block de command continu uit na elke delay tick zolang er 'stroom' is van redstone

Bij condition kan je kiezen tussen unconditional en conditional.
Bij conditional voert de command block de command alleen uit als de command klopt, dus als er een typfoutje in je command zit, dan voert de command block dat niet uit.
Bij unconditional voert de command block de command altijd uit, ookal is de command niet correct

Bij redstone kan je kiezen of je redstone 'stroom' nodig hebt om de command block te activeren, of je kan ervoor kiezen dat de command block altijd actief is

Bij delay in ticks kan je invoeren hoeveel game ticks er tussen elke uitvoering zit. "

3 zorg ervoor dat je agent blokken plaatst als hij beweegt
4 selecteer een blok dat je je agent wilt laten plaatsen
5 maak een herhaling waar je variabel hoogte keer herhaalt (

hoe bouw je een toren met agent

Kaart voor Gevorderden (meer dan 10 uur ervaring), gemaakt door Veda Rooze over Programmeren met blokjes

in deze lesbrief kom je er achter hoe je met je agent een toren bouwt in 10 simpele stappen.

"1 begin met een blok om een chatopdracht in toevoeren (geef deze een naam bijvoorbeeld tower) dit voer je later in de chatbalk om de opdracht in werking te zetten.
2 Teleporteer je agent naar je speler.
3 voer in at je agent vooruit beweegt met je gewenste aantal stappen.
4 voor in het blok waar agent stel in actief vak 1
5 zorg dat de agent tijdens het verplaatsen obstakels kan vernietigen. dit doe je door waar aan te vinken.
6 zorg dat je agent dit je gewenste aantal keer dit herhaalt.
7 zorg dat de agent een blok plaatst. kies je gewenste blok.
8 zorg dat je agent het aantal stappen wat je bij 3 hebt ingevoerd min 1 vooruit gaat.
9 laat je agent naar links draaien.
10 zorg dat je agent 1 omhoog beweegt.

Voer nu je ingevoerde naam in die je de opdracht hebt gegeven in de chat. je agent voert deze opdracht nu uit!"
